Razor Syntax

three basic rules 
	- just write code and markupe don't think about it
	- @ indicates code
	- standard language keyword
		e.g : if else foreach

Razor Keywords
	Code 
		@
		@()
		@{}
	Content
		@:
		<text></text>
	Comments
	- @* *@
